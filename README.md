# 🛡️ ZAKI - Link Checker

**ZAKI AI Link Checker** هو نظام متطور لفحص الروابط يعتمد على الذكاء الاصطناعي (Gemini 2.5 Flash) وتقنيات التحليل السيبراني لاكتشاف روابط التصيد الاحتيالي (Phishing) والروابط المشبوهة بدقة عالية.

---

## 🚀 Features | المميزات

- 🤖 **AI Analysis**  
  يستخدم نموذج Gemini 2.5 Flash لتحليل الروابط وفهم سلوكها الأمني.

- 🔗 **Hybrid Detection System**  
  دمج بين محركات فحص عالمية + ذكاء اصطناعي خاص بـ ZAKI لزيادة الدقة.

- 🌐 **Dynamic Favicon**  
  عرض أيقونة الموقع تلقائياً بجانب الرابط لتسهيل التعرف البصري.

- 📊 **Confidence Score**  
  تقييم أمان الرابط بنسبة مئوية من 0% إلى 100%.

- 🎨 **Futuristic UI**  
  واجهة حديثة (Dark Mode) مبنية بـ:
  - React.js  
  - Tailwind CSS  
  - Framer Motion  

- 🔒 **Zero Trust Logic**  
  رفض أي روابط أو مدخلات غير صالحة قبل معالجتها.

---

## 🛠️ Tech Stack | التقنيات المستخدمة

- React.js ⚛️ (Frontend)
- Tailwind CSS 🎨 (Styling)
- Framer Motion 🎬 (Animations)
- Lucide React 🎯 (Icons)
- Gemini 2.5 Flash API 🤖 (AI Security Engine)

---

## 📦 Installation | طريقة التشغيل

```bash
# Clone repository
git clone https://github.com/zg22x/LINK-CHECKER.git

# Enter project folder
cd link-checker

# Install dependencies
npm install lucide-react framer-motion

# Run project
npm start
```
